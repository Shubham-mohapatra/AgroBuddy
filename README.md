


AgroBuddy is an AI-powered mobile app (currently in development) aimed at helping farmers detect crop diseases and get tailored recommendations for treatment. Built using React Native, the app provides features like crop disease detection, weather integration, and offline functionality.

Features
Crop Disease Detection: Upload images of crops to detect diseases and get treatment suggestions.
Weather Integration: Provides weather-based planting and harvesting advice.
Offline Functionality: Works seamlessly in low-connectivity areas.

Tech Stack
Frontend: React Native
AI/ML: TensorFlow/Keras for disease detection
Backend: Python (Flask/Django)
